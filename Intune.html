About Intune

I have deployed Intune from scratch to manage our Windows and Apple devices. The deployment involved planning, configuring and testing all settings, policies and scripts.
Reviewing GPOs to figure out which of them are still relevant and need to be migrated to Intune.
Configuring, packaging and testing software for remote installations.
Remediation scripts to remove MS default apps, customise the GUI of Windows (Taskbar, Start menu, hide Cortana icon from taskbar) and apps by modifying registry keys using PowerShell (make "From:" field visible on Outlook) and to also do other admin tasks on a schedule (temp folder cleanup)
Windows Update policies and Autopatch, compliance policies, configuration profiles (deploy AOVPN xml profile, config MS Edge settings, restricting access to removable media USB, deploy custom Start/Taskbar menu layouts with pinned Apps)
Autopilot (hybrid/cloud), LAPS, troubleshooting enrolment errors (Conditional Access prevented our admin accounts enrolling devices unless we authenticated with MFA so I figued this out and configured an exception for Intune on our Conditiona Access policy)
